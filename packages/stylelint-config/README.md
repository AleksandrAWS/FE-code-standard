## 使用

### 安装

#### yarn
```shell
yarn add stylelint @provincial-platform/stylelint-config -D
```

#### npm

```shell
npm install stylelint @provincial-platform/stylelint-config --save-dev
```


### 配置

.stylelintrc
```text
{
  "extends": [
    "@provincial-platform/stylelint-config"
  ]
}
```
